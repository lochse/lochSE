# 使用CocosCreator开发的仿“魔塔”手游
cocos creator版本 v2.0.9

这是我们小组在大二下“软件工程”课程设计的项目

## 已实现的功能
- 虚拟摇杆控制移动 
  - onTouchCtrlUpdate.js控制角色移动
  - rocker.js控制摇杆移动
  - 撞墙会有几帧的贴边走 
- 伪场景切换 (portal.js)
  - 切换镜头的世界坐标来实现
  - 缺陷：开局预加载所有场景，如果场景过多可能会卡。
  - 优化方向：用二维数组记录每一关的每一个的组件，在切换时根据数组加载
